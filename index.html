Preguntas & Respuestas

¿Cómo se detectan los vehículos?

Para la detección de vehículos utilizamos un sensor inductivo, que con un campo electromagnético es capaz de sensorizar la interferencia del metal de los automóviles, motocicletas o bicicletas. El sensor que vamos a utilizar es el Lj12a3-4

¿Cómo manejamos la señal del sensor?

Para esta tarea utilizamos un Atmega328P, que es un circuito integrado de alto rendimiento, basado en un microcontrolador RISC (computador conjunto de instrucciones reducidas), con una memoria de 32KB flash con capacidad de leer entradas mientras esté ejecutando alguna acción en la parte de las salidas.

¿Cómo unimos todo?

Con una serie de componentes necesarios para que el Atmega328P funcione de manera adecuada, elementos de conexión, luces LED y una fuente para alimentar cada parte del sistema.
